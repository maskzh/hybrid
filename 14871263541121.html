预览图片

requestHybrid({
  action: 'previewImage', // action
  params: {
    current: '', // 当前显示图片的http链接
    urls: [],  // 需要预览的图片 http 链接列表
  }
})

参数

action: previewImage
params:

参数 值 类型 备注
current http://img.jkbsimg.com/up/fefe.png string 当前显示图片的http链接
urls [imgurl, imgurl, imgurl] array 需要预览的图片 http 链接列表
