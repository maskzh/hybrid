# 获取用户信息

```javascript
requestHybrid({
  action: 'getUserInfo',
  callback: function(res) {
    if (res.result) {
      var user = res.data
    }
  }
})
```
### 参数
---
action: **getUserInfo**
params: null

---
### 返回

```javascript
{
    result: true, // or false
    data: {
        name: '123',
        mobile: '18768178036',
        ...others
    },  // 用户信息
    message: 'false 提示文本，true 返回空'
}
```
