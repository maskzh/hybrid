# 获取地理位置

```javascript
requestHybrid({
  action: 'getLocation',
  callback: function(res) {
    if (res.result) {
      var lat = res.data.lat // 纬度
      var lon = res.data.lon // 经度
    }
  }
})
```
### 参数
---
action: **getLocation**
params: null
callback: function

---
### 返回

```javascript
{
    result: true, // or false
    data: { lat : '123', lon: '123', ...others }, // 经纬度
    message: 'false 提示文本，true 返回空'
}
```
